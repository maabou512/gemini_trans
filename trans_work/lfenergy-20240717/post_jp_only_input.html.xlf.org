<target xml:lang="ja">フランスの送電系統運用者（TSO）であるRTEは、エネルギー移行とデジタル<x id="1"/>変革を加速させ、未来の制御システムを構築しています。そのためには、電力網を変革するための新しい技術の開発と導入、まずは複数のプロトコル変換の問題を解決することが必要です。</target>
<target xml:lang="ja">RTEとそのパートナーであるDianomicは、産業用IoT <g id="3">LF Edge</g> プロジェクトである<g id="4">Fledge</g>をベースとした<g id="2">LF Energy</g>の電力システム向けマルチプロトコル変換ゲートウェイである<g id="1">FledgePOWER</g>がこの問題の解決に使用できることを確認しました。</target>
<target xml:lang="ja"><g id="1">FledgePOWER</g>は、電力システムのプロトコル変換とユースケースに重点を置いた、シミュレーション、データ構成、およびチェックのためのツールボックスを提供します。<g id="2">IEC 61850</g>は、変電所のインテリジェント電子デバイス用の通信プロトコルを定義した国際標準であり、物理的な通信層を抽象化し、これらのデバイスの相互運用性を確保するように設計されました。</target>

<target xml:lang="ja">FledgePOWERは、RTEやオランダの配電系統運用会社Allianderなど、さまざまなエネルギー関係者によって開発されたオープンソースツールであり、<g id="1">scanner</g>やクラウド上のデータセンター、変電所ネットワークなどの外部システムとのIEC 61850通信をサポートしています。</target>
